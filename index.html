Compromiso de Transparencia

Políticas claras para una experiencia confiable y segura

Términos de Servicio

  • Edad mínima: Debes tener al menos 13 años para realizar compras en nuestra plataforma
  • Cuentas válidas: Solo procesamos órdenes para cuentas de juego legítimas y personales
  • Uso personal: Todos los productos adquiridos son para uso exclusivo del comprador
  • Verificación: Nos reservamos el derecho de verificar transacciones para prevenir fraudes
  • Modificaciones: Los términos pueden actualizarse para mejorar la experiencia del usuario
  • Jurisdicción: Todas las transacciones se rigen bajo las leyes de Guatemala

Política de Reembolsos

  • Productos digitales: No aceptamos reembolsos una vez entregado el código o producto digital
  • Errores en compra: Contacta nuestro soporte dentro de las primeras 2 horas para asistencia
  • Problemas técnicos: Garantizamos la solución de cualquier issue dentro de 24 horas
  • No entregado: Reembolso completo si no entregamos tu producto dentro de 6 horas
  • Método de pago: Los reembolsos se realizan exclusivamente por el mismo método de pago original
  • Procesamiento: Los reembolsos toman entre 3-5 días hábiles en procesarse completamente

Política de Privacidad

  • Datos protegidos: Nunca compartimos información personal con terceros no autorizados
  • Transacciones seguras: Utilizamos encriptación SSL de 256-bit para todas las transacciones
  • Información mínima: Solo recolectamos datos estrictamente necesarios para el servicio
  • Comunicaciones: Solo contactamos para confirmaciones de compra y soporte al cliente
  • Cookies: Empleamos cookies esenciales únicamente para funcionalidad del sitio web
  • Derechos del usuario: Puedes solicitar la eliminación de tus datos en cualquier momento
